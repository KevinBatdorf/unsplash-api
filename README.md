# Unsplash API

Set your access key to `UNSPLASH_ACCESS_KEY` in `.env`

## Get Photos

List 10 photos ([See it](https://unsplash-api-search.vercel.app/api/photos))
```
/api/photos
```

Search by query ([See it](https://unsplash-api-search.vercel.app/api/search/photos))
```
/api/search/photos
```
## Terms
If using this API directly instead of cloning it, you should read these terms of use: [terms.html](https://unsplash-api-search.vercel.app/terms.html). In short, clone this repo and host it on Vercel yourself rather than use this API outright. This code is emant to be used with the [Unlimited Photos](https://wordpress.org/plugins/unlimited-photos) WordPress block plugin.
